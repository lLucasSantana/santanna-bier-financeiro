# 🍺 Santanna Bier - Sistema Financeiro

Sistema financeiro desenvolvido para a empresa **Santanna Bier** como Projeto Integrador do curso de **Análise e Desenvolvimento de Sistemas** da **URI Erechim**.

---

## 📖 Sobre o Projeto

O objetivo deste sistema é auxiliar no gerenciamento financeiro da empresa, oferecendo uma solução prática, organizada e intuitiva para controle financeiro empresarial.

O sistema permite realizar:

- ✅ Cadastro de informações financeiras
- ✅ Controle de entradas e saídas
- ✅ Gerenciamento de lançamentos
- ✅ Visualização de resultados financeiros
- ✅ Organização de despesas e receitas
- ✅ Melhor acompanhamento financeiro da empresa
